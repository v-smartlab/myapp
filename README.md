## Quick Start

```bash
# Development
docker compose up --build

# Staging
APP_ENV=staging docker compose up --build

# Production
APP_ENV=prod APP_VERSION=1.0.0 docker compose up -d
```

## Build & Push

```bash
chmod +x build-push.sh
./build-push.sh prod 1.0.0
```
